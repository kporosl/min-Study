# C 기본

```
정수(일반 숫자) : int , long , short , long long
실수(소숫점) : float , doube , long double  
문자열 : char = 'a'
8 bit = 1 byte
1 bit = 0.125 byte

정수
int : 4byte
short : 2byte
long : 4byte
long long : 8byte

실수 
float : 4byte
double : 8byte
long double : 12byte

문자열
char : 1byte

* : 곲
/ : 나눔
- : 빼기
+ : 더하기
% : 나머지 

%c : 문자형 단일 문자(char 형 변수)를 출력합니다.
%s : 문자열 문자열(char 형 배열)을 출력합니다.

'' : 단일 문자(character)를 나타냅니다.
"" : 문자열(string)을 나타냅니다.
```
