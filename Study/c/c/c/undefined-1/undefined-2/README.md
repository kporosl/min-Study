C언어는 0이 아닌 모든 값을 참으로 간주한다. 다만 1이 참을 의미하는 대표적인 값일 뿐이다.

연산의 조건을 만족하면 참을 의미하는 1을 반홖하고 만족하지 않으면 거짓을 의미하는 0을 반홖하는 연산 자들이다.
